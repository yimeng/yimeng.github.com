mac tools

这个博客写点关于技术相关,好不容易过年的7天有点时间.第一篇写写mac上遇到的各种软件和小技巧吧.

##工具

###brew
使用方法啥的估计一搜一大堆,这里不多说了.这里说说一些注意事项:

  • 安装软件之前brew update下,不然可能有一些软件的依赖可能会出现意想不到得小问题
  • 如果团队使用mac同学比较多,可以自定义用brew tap定义一个团队的formula给大家用
  • 如果自己对命令行有这特殊的感情,可以加装http://caskroom.io 不过个人感觉,安装软件是个乐趣.没必要那么geek.

写作

Mou

写写技术文档,可以用这个.前一阵子开发者甚至要把这个软件出售.不过后来好了.还能继续更新.

不过Mou1.0好像要卖180软妹币.现在正在众筹90软妹币,果断购入了.不仅是支持国内的程序员.也是对Mou这个软件的肯定.但是不知道Mou1.0有什么特性.2015年4月份出来.届时在看呗.

官方地址

2015年6月6日更新 这家伙跳票了 v2ex上声讨的声音比较大 哎 就当给国产软件做贡献了吧

###Byword
对于要专注写作但是又不是长篇大论严肃的同学.但应该不太适合写技术文档?

Markdown Plus

秦老板 在微博上推荐的. 说实话,能画流程图和对于各种markdown的支持吸引了我.果断抛弃了Mou,投奔这里.不过新软件,有些功能确实还需要继续完善.比如一次只能打开一个文档(也可能是有意为之?).
